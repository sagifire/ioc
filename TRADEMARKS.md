# Trademarks

`@sagifire/ioc` is the product mark for this library family, including the related package
names `@sagifire/ioc-next` and `@sagifire/ioc-testing`.

This policy documents project mark usage expectations. It does not claim that the mark is a
registered trademark.

customary use in describing the origin of the work and reproducing the NOTICE file.

## Allowed Uses

You may use the `@sagifire/ioc` name to:

- Refer truthfully to the original project or packages.
- Identify that your project depends on or is compatible with `@sagifire/ioc`.
- Link to the original repository, documentation or package metadata.
- Preserve required copyright, license, trademark and attribution notices.
- Describe unmodified redistributed copies of the project or packages.

## Reserved Uses

Do not use the `@sagifire/ioc` mark in a way that suggests sponsorship, endorsement or
official status unless that permission has been granted by the project maintainers.

Reserved uses include:

- Naming a modified redistribution so that it appears to be the original project.
- Using a confusingly similar package, project, domain, organization or product name.
- Presenting an integration, plugin, fork, template or service as official when it is not.
- Removing or rewriting required NOTICE or trademark guidance in redistributed copies.

## Forks And Modified Versions

Forks and modified versions should use a distinct name unless they are clearly presented as
unmodified copies or as descriptive compatibility work. It is fine to say that a fork is
based on `@sagifire/ioc`, but the fork name should not imply that it is the official
project.

## Questions

Use GitHub Issues for ordinary product mark questions that are not security-sensitive.
