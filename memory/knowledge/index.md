# Індекс: knowledge

## Призначення

Reusable Knowledge Memory для знань, які мають застосовуватися між задачами, сесіями або проектами.

Knowledge Memory не є основним startup layer. На старті агент читає `memory/agent-start.md`, а до knowledge packages переходить тільки тоді, коли задача або `memory/knowledge/package-index.md` показують релевантний пакет.

## Папки

- [Packages](packages/index.md) - Knowledge packages, включно з `pdadm-mvp-reglament/`.
- [Adaptations](adaptations/index.md) - Адаптації зовнішніх або переносимих знань під цей проект.
