# Human Start

Starter Kit Version: 4.0
PDADM MVP Version: 0.4

## Для чого цей файл

Короткий вхід у Project Memory для людини, яка ще не знайома з PDADM MVP.

## Як попросити агента допомогти

```text
Agent Role: Agent Assistant
Working Mode: Methodology Navigator
```

## Перші кроки

1. Подивитися `memory/state.md`.
2. Якщо проект новий, пройти Project Bootstrap.
3. Якщо проект уже має задачі, відкрити `memory/tasks/plan/progress.md`.
4. Якщо потрібні зміни в коді або Project Memory, створити або уточнити задачу.
5. Після роботи агента перевірити self-review і явно прийняти або повернути результат.
