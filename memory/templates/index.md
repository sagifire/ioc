# Індекс: templates

## Призначення

Шаблони для startup protocol, задач, autonomous implementation runs, autonomous research, detailed reports, audits, interactive fixations, ADR і knowledge packages.

## Папки

Немає дочірніх папок.

## Файли

- [Agent Start](agent-start.md) - Шаблон startup entrypoint для агентів.
- [Human Start](human-start.md) - Шаблон human-facing entrypoint для користувача.
- [Task](task.md) - Шаблон task contract.
- [Run Requirements](run-requirements.md) - Шаблон вимог конкретного autonomous implementation run.
- [Context](context.md) - Шаблон context package для autonomous implementation run.
- [Result](result.md) - Шаблон result report для autonomous implementation run.
- [Research Report](research-report.md) - Шаблон task-local `RSCH-*` для `autonomous-research`.
- [Research Detailed Report](research-detailed-report.md) - Шаблон detailed report для `reports/research/**`.
- [Audit Report](audit-report.md) - Шаблон audit report для `reports/audits/**`.
- [Worklog](worklog.md) - Шаблон worklog для interactive memory update.
- [Fixation](fixation.md) - Шаблон fixation package для контрольованих змін Project Memory.
- [ADR](adr.md) - Шаблон ADR.
- [Knowledge Package](knowledge-package.md) - Шаблон knowledge package.
- [Knowledge Example](knowledge-example.md) - Шаблон прикладу для knowledge package.
- [Knowledge Anti-Example](knowledge-anti-example.md) - Шаблон anti-example для knowledge package.
