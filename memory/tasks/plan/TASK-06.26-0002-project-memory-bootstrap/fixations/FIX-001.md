істини. Project Memory має містити операційний, структурований і навігаційний контекст,
щоб Stage 2 repository/build foundation і наступні stages виконувались проти однакових
архітектурних меж, acceptance criteria і DoD.

project-specific, а не переносимим знанням між проектами.
